                              
                             Как пользоваться активатором?

На рабочем столе находится папка Активация зайдите в неё и запустите файл "Windows Loader.exe" он же Windows Loader by Daz кликните по нему мышью откроется окно затем нажмите на кнопку "Install" после дождитесь подтверждения для перезагрузки компьютера.
Системные требования:

Microsoft Windows 7 / Vista / 2008 R2 / Server 2012


Описание:

Windows Loader v2.2.2 Это загрузочное приложение, используемое миллионами людьми по всему миру, хорошо известное тем, что обходит Microsoft's WAT (Windows Activation Technologies) и, вероятно, является самым безопасным из всех подобных. Приложение внедряет SLIC (System Licensed Internal Code) в вашу систему до начала загрузки Windows, да так, что "недалёкая Windows" начинает считать себя активированной.

Активатор Microsoft Windows 7 / Vista / 2008 R2 / Server 2012, устанавливает сертификат одной из фирм (по выбору), после чего активирует вашу копию Windows и дает возможность проходить проверку подлинности. Запускать от имени Администратора (желательно из проводника Windows)
